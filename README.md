# 📱 政治情報収集ナビゲーションアプリ (Android)

## 🎯 プロジェクト概要
「若者の政治離れ」という社会課題に対し、分散している政治情報を手軽に一元的に収集・閲覧できるAndroid専用アプリです。
情報の取得ハードルを下げることで、若年層が政治に関心を持つきっかけを作ることを目的として企画・開発しました。

*※本プロダクトはサーバー等の運用コストの観点からストアへの正式リリースは見送りましたが、要件定義からプロトタイプ完成までの全工程を走り切った実証開発プロジェクトです。*

## 🛠 技術スタック
* **開発言語:** Java
* **開発環境:** Android Studio
* **フレームワーク:** Android SDK

## 👥 開発体制と役割
* **チーム構成:** 2名（研究室の先輩と共同開発）
* **担当範囲:** 要件定義、UI/UX設計、実装、テストまでの全工程
* **開発スタイル:** 明確な分業（フロント/バックエンド等の切り分け）はあえて設けず、継続的にアイデアをぶつけ合い、試行錯誤を繰り返しながら全フェーズを2人で共同推進しました。

## 💡 こだわったポイント：徹底したユーザー目線でのUI/UX改善
本開発において最も直面した壁であり、同時に最も注力したのが**「ユーザーにとって本当に使いやすいUI/UXの実現」**です。

1. **直面した課題（機能過多による複雑化）:**
   初期プロトタイプでは機能の実装を優先した結果、画面上の情報量が過多になり、ターゲットである若者にとって直感的な操作が難しい状態に陥ってしまいました。
2. **解決へのアプローチ（対話力を活かしたヒアリング）:**
   「徹底した利用者目線」に立ち返るため、ターゲット層と同世代の学生に実際にアプリを触ってもらうユーザーテストを実施しました。ヒアリングの際は、部活動の対立解消などで培った「相手の本音を引き出す対話力」を応用し、ユーザー自身も言語化できていない潜在的な使いにくさやニーズを抽出しました。
3. **改善策と結果:**
   得られた生の声をもとに、情報の階層化（グルーピング）と画面遷移の導線を徹底的にシンプルに改修。結果として操作性が大幅に向上し、当初の目的である「手軽な情報収集」を実現するUIへと進化させることができました。

## 🎓 プロジェクトを通して得た学び
本開発を通して、自己満足の技術実装で終わらせず、ユーザーの生の声を聞き**「確かなユーザー価値へと変換する設計」**の重要性を痛感しました。また、社会課題を自分なりに捉え、それをテクノロジーを用いてゼロから形にする「企画力」と「推進力」を培うことができました。
